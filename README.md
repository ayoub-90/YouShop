# YouShop

Youshop is a comprehensive ecommerce website built using the Laravel framework, JQuery, HTML, CSS, and Bootstrap. It is a user-friendly platform designed to provide a seamless and intuitive shopping experience for customers.

The website features a range of product categories, including electronics, fashion, beauty, home goods, and more. Users can browse through the products, add items to their shopping cart, and checkout with ease. The site also offers a search feature to help customers find specific products quickly.

To enhance the customer experience, Youshop offers a personalized account system. Users can create an account to store their personal information, view their order history, and manage their shopping cart. The site also offers a wishlist feature, which allows customers to save items for later.

The Youshop ecommerce website is fully responsive and optimized for mobile devices, ensuring that customers can easily access the site from their smartphones or tablets. The design is clean and modern, with an emphasis on functionality and usability. The site is also optimized for fast loading times, ensuring that customers can quickly find what they're looking for and complete their purchase.

To make payments secure and easy, Youshop integrates with various payment gateways such as PayPal, Stripe, and credit/debit card payments. making it accessible to a global audience.

Overall, Youshop is a reliable and efficient ecommerce website, providing a seamless shopping experience for customers while offering an easy-to-use platform for sellers to manage their products and transactions.

# Getting started

* Clone the repo : 
```shell
git clone https://github.com/ayoub-90/YouShop.git
```

* Start the SQL server.

* Start Laravel development server :
```shell
php artisan serve
```
Now, the app is running at http://127.0.0.1:8000 !
